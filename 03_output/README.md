# Outputs

This directory is for all generated and output files related to the research project.

This includes:

- The presentation `Basic_diff_in_diffs.qmd`
   - renders to html reveal.js and explains key difference-in-difference concepts using examples from [Chapter 18](https://theeffectbook.net/ch-DifferenceinDifference.html) of Nick Huntington-Klein's book [The Effect](https://theeffectbook.net), which is available for free online on <https://theeffectbook.net>



- Figures (as `.pdf`) and tables  (as `.md`) related to replicating Colantone et al. (2023) 

- `Comparison.qmd`, which renders to a html document
   - this uses `references.yaml` to format the Colantone et al. (2023) citation
